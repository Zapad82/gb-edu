Секундомер



В решении данной задачи использованы следующие парадигмы программирования:

  1. Объектно-ориентированное программирование (ООП): создан класс Stopwatch, который инкапсулирует состояние и поведение секундомера. Все методы для управления секундомером определены внутри класса.

  2. Императивное программирование: в основе решения лежит последовательное выполнение команд. Код содержит инструкции для выполнения определенных действий при нажатии на кнопки.

  3. Событийно-ориентированное программирование: функции startStopwatch(), pauseStopwatch(), resumeStopwatch() и stopStopwatch() вызываются в ответ на события "onclick" кнопок. Это позволяет связать определенные действия с определенными событиями пользователя.
