    <h1> Основы CSS </h1>
    
    <h3> 1. Что такое css? </h3>
    Это каскадная таблица стилей.<br> Формальный язык описания внешнего вида документа, написанного с использованием языка
    разметки.<br>
    Преимущественно используется как средство описания, оформления внешнего вида веб-страниц.<br><br>

    <h3>2. Синтаксис css?</h3>
    Селектор {<br>
    свойство1:значение1;<br>
    свойство2:значение;<br>
    }<br>
    Пример:<br>
    p {<br>
    color:blue;<br>
    }<br><br>

    <h3>3. Оформление стилей</h3>
    Селектор {<br>
    свойство1:значение1;<br>
    свойство2:значение;<br>
    }<br>
    Самый рациональный способ оформления, когда Селектор открывает свойства на одной строке, все остальное ниже
    строками.<br><br>

    <h3>4. Комментарии</h3>
